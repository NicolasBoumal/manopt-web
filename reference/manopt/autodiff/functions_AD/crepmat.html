crepmat

PURPOSE

Replicates an array.

SYNOPSIS

function Xrepmat = crepmat(X, varargin)

DESCRIPTION 

 Replicates an array.

 function Xrepmat = crepmat(X,varargin)

 This function can be seen as repmat(X,varargin) but is compatible with
 structs with fields real and imag.
