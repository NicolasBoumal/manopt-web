truncate

PURPOSE

ROUND Approximate TTeMPS tensor within a prescribed tolerance.

SYNOPSIS

function x = round( x, r )

DESCRIPTION 

ROUND Approximate TTeMPS tensor within a prescribed tolerance.
   X = ROUND( X, tol ) truncates the given TTeMPS tensor X to a
   lower rank such that the error is in order of tol.
