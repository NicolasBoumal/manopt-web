round

PURPOSE

ROUND Approximate TTeMPS operator within a prescribed tolerance.

SYNOPSIS

function A = round(A, tol )

DESCRIPTION 

ROUND Approximate TTeMPS operator within a prescribed tolerance.
   X = ROUND( A, tol ) truncates the given TTeMPS operator A to a
   lower rank such that the error is in order of tol.
