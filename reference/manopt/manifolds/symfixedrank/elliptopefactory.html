elliptopefactory

PURPOSE

Manifold of n-by-n PSD matrices of rank k with unit diagonal elements.

SYNOPSIS

function M = elliptopefactory(n, k)

DESCRIPTION 

 Manifold of n-by-n PSD matrices of rank k with unit diagonal elements.

 function M = elliptopefactory(n, k)

 The geometry is based on the paper,
 M. Journee, P.-A. Absil, F. Bach and R. Sepulchre,
 "Low-Rank Optimization on the Cone of Positive Semidefinite Matrices",
 SIOPT, 2010.

 Paper link: http://www.di.ens.fr/~fbach/journee2010_sdp.pdf

 A point X on the manifold is parameterized as YY^T where Y is a matrix of
 size nxk. The matrix Y (nxk) is a full column-rank matrix. Hence, we deal
 directly with Y. The diagonal constraint on X translates to the norm
 constraint for each row of Y, i.e., || Y(i, :) || = 1.
 
 See also: obliquefactory

0118 function etaproj = projection(Y, eta)
0119     [unused, k] = size(Y); %#ok<ASGLU>
0120     eta = project_rows(Y, eta);
0121 
0122     % Projection onto the horizontal space
0123     YtY = Y'*Y;
0124     SS = YtY;
0125     AS = Y'*eta - eta'*Y;
0126     try
0127         % This is supposed to work and indeed return a skew-symmetric
0128         % solution Omega.
0129         Omega = lyap(SS, -AS);
0130     catch %#ok<CTCH> Octave does not handle the input of catch, so for
0131         % compatibility reasons we cannot expect to receive an exception object.
0132         % It can happen though that SS will be rank deficient. The
0133         % Lyapunov equation we solve still has a unique skew-symmetric
0134         % solution, but solutions with a symmetric part now also exist,
0135         % and the lyap function doesn't like that. So we want to
0136         % extract the minimum norm solution. This is also useful if lyap is
0137         % not available (it is part of the control system toolbox).

0140         is_octave = exist('OCTAVE_VERSION', 'builtin');
