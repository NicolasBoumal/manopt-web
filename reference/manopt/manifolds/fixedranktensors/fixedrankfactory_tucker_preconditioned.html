fixedrankfactory_tucker_preconditioned

PURPOSE

Manifold of fixed-multilinear-rank tensors.

SYNOPSIS

function M = fixedrankfactory_tucker_preconditioned(tensor_size, tensor_rank)

DESCRIPTION 

 Manifold of fixed-multilinear-rank tensors.

 function M = fixedrankfactory_tucker_preconditioned(tensor_size, tensor_rank)

 n1 = tensor_size(1);
 n2 = tensor_size(2);
 n3 = tensor_size(3);
 r1 = tensor_rank(1);
 r2 = tensor_rank(2);
 r3 = tensor_rank(3);

 A point X on the manifold is represented as a structure with four
 fields: U1, U2, U3 and G. 

 The matrices U1 (n1-by-r1), U2 (n2-by-r2) and U3 (n3-by-r3) are orthogonal
 matrices.
 G (r1-by-r2-r3) is a multidimensional array.

 Tangent vectors are represented as a structure with four fields: U1, U2, U3
 and G.

0211         
0212         % First, projection onto tangent space of total space

0215         BU1 = lyap(SSU1, -ASU1);

0217         
0218         SSU2 = X.G2G2t;
0219         ASU2 = 2*symm(X.G2G2t*(X.U2'*eta.U2)*X.G2G2t);
0220         BU2 = lyap(SSU2, -ASU2);
0221         eta.U2 = eta.U2 - X.U2*(BU2/X.G2G2t);
0222         
0223         SSU3 = X.G3G3t;
0224         ASU3 = 2*symm(X.G3G3t*(X.U3'*eta.U3)*X.G3G3t);
0225         BU3 = lyap(SSU3, -ASU3);
0226         eta.U3 = eta.U3 - X.U3*(BU3/X.G3G3t);
0227         
0228 
0229         eta_G1 = reshape(eta.G, r1, r2*r3); 
0230         eta_G2 = reshape(permute(eta.G, [2 1 3]), r2, r1*r3); 
0231         eta_G3 = reshape(permute(eta.G, [3 1 2]), r3, r1*r2);
0232         
0233         
0234         % Project onto the horizontal space.
0235         PU1 = skew((X.U1'*eta.U1)*X.G1G1t) + skew(X.G1*eta_G1');
0236         PU2 = skew((X.U2'*eta.U2)*X.G2G2t) + skew(X.G2*eta_G2');
0237         PU3 = skew((X.U3'*eta.U3)*X.G3G3t) + skew(X.G3*eta_G3');

0240         % horizontal component. Currently, we are using a simple iterative
0241         % technique to solve the system of equations.
0242         %
0243         % We use the Matlab's pcg function to solve the system efficiently.
0244         % We exploit the structure by designing a good preconditioner as well.
0245         % The preconditioner takes the block positive definite part of the
0246         % linear system.

0248         % Options for PCG
0249         tol_omegax_pcg = 1e-6; % BM: standard tolerance as suggested in PCG.

0254         M2 = kron(speyer2,SSU2) + kron(SSU2, speyer2);
0255         M3 = kron(speyer3,SSU3) + kron(SSU3, speyer3);

0259         Mprecon_pcg(1 + r1^2 : r1^2 + r2^2, 1 + r1^2 : r1^2 + r2^2) = M2;
